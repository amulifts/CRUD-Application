# Address Book

A simple CRUD application using HTML, CSS and JavaScript completed as a mini project.

## Usage

```powershell
    # For Windows
    start abook.html
```
```powershell
    # For MacOS
    open abook.html
```
```powershell
    # For Linux
    xdg-open abook.html
```

## Features

- Create, read, update and delete records (CRUD)
